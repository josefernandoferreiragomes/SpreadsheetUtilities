---
name: second-opinion
description: Get a second opinion on code from another agent/model
---

**Instructions**:

1. Come up with a clearly defined and worded request for another AI agent on something you want advice or help with.

2. Use the following cli command to ask for advice / request the another AI agent.
```
opencode run "<REQUEST>" -m opencode/ring-2.6-1t-free
```

**Parameters**:
- `<REQUEST>`: Replace this with your request for another AI agent.

3. Potentially wait a while while the model is processing your request and generating a response.

4. Communicate concisely to me what the other model thought and how you think about it
